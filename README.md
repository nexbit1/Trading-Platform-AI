# AI Trading Floor 🤖📈

A multi-agent AI trading simulation where autonomous AI traders research markets, analyze strategies, manage portfolios, and execute trades using MCP (Model Context Protocol).

## Features

- 🤖 Multiple AI traders with different strategies/personas
- 🧠 Multi-model support (OpenAI + Groq)
- 🔍 AI-powered market research using MCP tools
- 📊 Portfolio tracking dashboard
- 🌐 FastAPI backend for frontend integration
- 💾 Persistent trader memory
- 📈 Simulated market trading environment

## Tech Stack

- Python
- OpenAI Agents SDK
- Model Context Protocol (MCP)
- FastAPI
- Gradio
- SQLite
- Plotly
- Groq / OpenAI APIs
- React / Vite
